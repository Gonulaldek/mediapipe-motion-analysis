# Reference Videos

This folder is used for local reference exercise videos.

Video files are not included in the repository to keep the project lightweight.

Expected local filenames:

- bicep_ref.mp4
- squat_ref.mp4
- finger_ext_ref.mp4
- finger_spread_ref.mp4
- wrist_flexion_ref.mp4

Place your own reference videos in this folder before running the project locally.
